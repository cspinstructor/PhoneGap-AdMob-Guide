10. Live ads

Download and install java then open a command line and issue this command:

keytool -genkey -v -keystore Test.keystore -alias Test -keyalg RSA -keysize 2048 -validity 10000

If you get the error saying keytool not found, you need to add the path to the java binaries to your "path" environment variable.

keytool -genkey -v -keystore Test.keystore -alias Test -keyalg RSA -keysize 2048 -validity 10000

Replace Test with your own preferred name. After the keystore is generated, upload it to build.phonegap.com following the steps in the picture below:

Next, edit your ads.js file to set "isTesting: false". This will fetch live ads from AdMob.
